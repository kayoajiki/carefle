## はじめに

### キャリフレとは

**キャリフレ**は、自己理解の旅を支援する**内省支援プラットフォーム**です。

診断・記録・行動計画のツールを通じて、あなたの「今」を可視化し、「これから」を具体化します。転職する／続ける／働き方を変える——まずは自分を理解することから始めましょう。

### キャリフレの目的・ビジョン

#### コア価値：「内省伴走」

キャリフレのコア価値は**「内省伴走」**です。

AIが親しい友人や信頼できる相談相手のように、あなたの内省を優しく伴走します。ユーザーの話を共感的に、注意深く聞き、深い内省を促す質問をします。励まし、前向きな気づきを引き出し、ユーザーの価値観や目標を尊重し、判断を下しません。

#### 目的

**悩み続ける時間を「進んでいる実感がある時間」に置き換える**

これがキャリフレの目的です。ただ悩み続けるのではなく、診断・記録・行動計画を通じて、自分自身を理解し、次の一歩を具体的に進める実感を提供します。

#### ビジョン

**日常的な自己理解の習慣が、あなたを変える**

キャリフレは、一時的な診断や相談ではなく、日常的に自分と向き合う習慣を身につけることを目指しています。日々の内省が積み重なって、長期的な目標に接続され、「道になる」体験を提供します。

#### コアKPI

- **サービス全体KPI**: 行動した、実行した、行動実感・成長実感
- **キャリフレ内KPI**: 内省の習慣化（日記を書く、振り返る頻度）

### 実現したいこと

#### 1. 自分で気づくツールを提供

キャリフレは、答えを与えるのではなく、**自分の中の答えを見えるようにする**ツールを提供します。

- 診断・記録・可視化のツールで、あなたの中の答えを見えるようにする
- 決めつけたり、特定の選択を押しつけたりはしない
- 自分で気づくための環境と仕組みを整える

#### 2. 「今のあなた」を否定しない

今の環境で耐えてきたことも、守ってきたものも、**ちゃんと意味がある**。

- 「がまんしてきた自分」を置き去りにしない
- 今の状況を否定するのではなく、そこから次の一歩を見つける
- あなたの価値観や選択を尊重する

#### 3. データとして蓄積され、可視化される

ふわっとした自己理解では終わらせません。

- あなたの診断結果・記録・行動計画がすべてデータとして蓄積される
- レーダーチャートやタイムラインで可視化される
- 悩んだときに確認できる、あなただけの自己理解マップ

#### 4. 内省が「道になる」体験

日々の内省が積み重なって、長期的な目標に接続される。

- 日々の内省と長期的目標の接続強化
- 内省の蓄積と活用
- 日記からマイルストーンへの自動接続
- 価値観の変化を追跡し、成長を実感できる

```
│  🎯 WCMシート・マイルストーン・アクションプラン            │

```

```
       │                    │                    │
       └────────────────────┴────────────────────┘
```

- **人生史**：タイムラインで価値観のルーツを発見
- **マイルストーン**：目標とアクションを管理

## 診断・分析機能

### 📊 現職満足度診断

現職満足度診断では、仕事と暮らしの満足度バランスを6つの領域で診断します。

#### 6つの領域

1. **Purpose（目的・意義）**：仕事の目的や意義に対する満足度
2. **Life（生活）**：ワークライフバランスや生活の質
3. **Privilege（特権・待遇）**：給与、福利厚生、待遇
4. **Position（地位・立場）**：役職、キャリアステージ、社会的地位
5. **People（人間関係）**：同僚、上司、部下との関係
6. **Performance（成果・評価）**：成果の出しやすさ、評価の公平性

#### 診断の流れ

1. **満足度の診断**：各領域について、現在の満足度を0-100点で評価
2. **重要度の診断**：各領域について、自分にとっての重要度を0-100点で評価
3. **結果の可視化**：レーダーチャートで満足度と重要度を比較表示
4. **スコア計算**：重み付け平均による正確なスコア計算（0-100点）
5. **動的コメント生成**：満足度と重要度の差分に基づいて、フォーカス領域と活かしたい強みを自動抽出

#### レーダーチャートの見方

レーダーチャートでは、6つの領域を6角形で表示します。

```
         Purpose
            /\
           /  \
          /    \
    People      Life
         |      |
         |      |
    Position    Privilege
         \      /
          \    /
           \  /
         Performance
```

- **満足度（青線）**：現在の満足度を表す
- **重要度（オレンジ線）**：自分にとっての重要度を表す
- **差分**：重要度が高く満足度が低い領域は、改善の余地がある領域
- **バランス**：6つの領域のバランスを一目で把握

**見方のポイント**：
- 青線がオレンジ線より内側にある領域 → 改善の余地がある
- 青線がオレンジ線より外側にある領域 → 満足度が重要度以上
- 6つの領域が均等に配置されている → バランスが取れている

#### スコアの意味

- **満足度スコア**：6つの領域の満足度の重み付け平均（重要度を重みとして使用）
- **重要度スコア**：6つの領域の重要度の平均
- **スコアが100点の場合**：すべての領域で満足度が重要度以上
- **スコアが低い場合**：重要度が高く満足度が低い領域がある

#### アクセス方法

- サイドバーから「現職満足度診断」を選択
- または、ダッシュボードから「診断を始める」をクリック
- ルート：`/diagnosis/start`

### 📊 自己診断結果

自己診断結果では、複数の診断タイプの結果を記録・管理・比較できます。

#### 対応診断タイプ

1. **MBTI（16タイプ）**：性格タイプを16種類に分類
2. **ストレングスファインダー（34の強み）**：あなたの強みを34種類から特定
3. **エニアグラム（9つのタイプ）**：性格を9つのタイプに分類
4. **ビッグファイブ（5因子）**：性格を5つの因子で分析
5. **FFS理論（5つの特性）**：凝縮性・受容性・弁別性・拡散性・保全性
6. **カスタム診断**：独自の診断結果を登録

#### 診断結果の登録方法

1. サイドバーから「自己診断結果」を選択
2. 「診断結果を登録する」をクリック
3. 診断タイプを選択
4. 診断結果を入力（タイプ名、スコア、詳細など）
5. 完了日を設定
6. 保存

#### 可視化機能

- **複数の診断結果をレーダーチャートで比較表示**
- 時系列で診断結果の変化を追跡
- 異なる診断タイプの結果を比較

#### アクセス方法

- サイドバーから「自己診断結果」を選択
- ルート：`/assessments`

### 📊 人生史

人生史では、人生のターニングポイントを時系列で可視化し、価値観のルーツを発見します。

#### タイムライン表示

- 人生の出来事を時系列で表示
- 各出来事にモチベーションスコアを記録
- モチベーショングラフで人生の浮き沈みを可視化

#### 出来事の追加方法

1. サイドバーから「人生史」を選択
2. 「出来事を追加」をクリック
3. 日付、タイトル、説明、モチベーションスコアを入力
4. 保存

#### 価値観の抽出機能

- 過去の出来事から価値観のルーツを発見
- AIが出来事を分析し、価値観のキーワードを抽出
- WCMシートのWillと連携

#### アクセス方法

- サイドバーから「人生史」を選択
- ルート：`/life-history/timeline`

## 記録・振り返り機能

### 📝 日記

日記では、毎日のモチベーションと出来事を記録し、内省を習慣化します。

#### カレンダービュー

- 月次カレンダーで過去の記録を振り返り
- 日記を書いた日は色が変わる
- 日付をクリックして日記を表示・編集

#### 日記の記入方法

1. サイドバーから「日記」を選択
2. カレンダーから日付を選択
3. モチベーションスライダーを調整（0-100）
4. 日記の内容を入力
5. 写真を追加（オプション）
6. 内省タイプを選択（オプション）
7. 保存

#### モチベーション記録

- 毎日のモチベーションを0-100で記録
- モチベーション推移グラフで可視化
- 週次・月次の平均モチベーションを表示

#### 写真の追加

- 日記に写真を添付
- 写真付きカレンダーで視覚的に振り返り

#### 内省タイプの分類

日記には以下の内省タイプを設定できます：

- **daily（1日の振り返り）**：今日の出来事を振り返る
- **yesterday（昨日の振り返り）**：昨日の出来事を振り返る
- **weekly（週次振り返り）**：1週間を振り返る
- **deep（深い内省）**：深く自分と向き合う
- **moya_moya（モヤモヤ解消）**：モヤモヤした気持ちを整理

#### アクセス方法

- サイドバーから「日記」を選択
- ルート：`/diary`

### 💬 内省チャット

内省チャットでは、AIとの対話型内省を通じて、深い内省を促します。

#### AIとの対話型内省

内省チャットは、**LINEスタイルのチャットUI**で、親しみやすい会話形式です。

```
┌─────────────────────────────────────────┐
│  ⭐ AIメッセージ（左側）                  │
│  今日はどんな1日でしたか？                │
│                                         │
│  ユーザーメッセージ（右側）               │
│  仕事で少し疲れました                    │
│                                         │
│  ⭐ AIメッセージ（左側）                  │
│  お疲れ様です。どんなことが疲れの原因     │
│  でしたか？                              │
└─────────────────────────────────────────┘
```

- **優しい伴走者**：AIが親しい友人や信頼できる相談相手のようにサポート
- **共感的な応答**：ユーザーの話を共感的に、注意深く聞く
- **深い内省を促す質問**：状況に応じた深い質問を生成
- **簡潔な応答**：1回の応答は2-3文程度で、自然な会話の流れを大切にする

#### トピック選択

内省を始めると、以下のトピックから選択できます。トピックは2列のグリッドレイアウトで表示されます。

```
┌──────────┐  ┌──────────┐
│  仕事    │  │  家族    │
└──────────┘  └──────────┘
┌──────────┐  ┌──────────┐
│  恋愛    │  │ 人間関係  │
└──────────┘  └──────────┘
┌──────────┐  ┌──────────┐
│  健康    │  │  目標    │
└──────────┘  └──────────┘
┌──────────┐  ┌──────────┐
│  学び    │  │  その他  │
└──────────┘  └──────────┘
```

- **仕事**：仕事に関する内省
- **家族**：家族に関する内省
- **恋愛**：恋愛に関する内省
- **人間関係**：人間関係に関する内省
- **健康**：健康に関する内省
- **目標**：目標に関する内省
- **学び**：学びに関する内省
- **その他**：その他の内省

#### 会話の保存

- 会話履歴は自動的に保存される
- 日記と連携して、会話内容を日記として保存
- 過去の会話を振り返り

#### 週次・月次振り返り

- **週次振り返り**：1週間の日記から、AIがサマリーと気づきを提供
- **月次振り返り**：1ヶ月の日記から、AIがサマリーと気づきを提供
- 成長の実感を可視化

#### アクセス方法

- ダッシュボードから「今日の内省を始める」をクリック
- または、サイドバーから「日記」→「内省チャット」
- ルート：`/diary/chat`

#### 過去の内省の検索

- 成長の実感を可視化

#### アクセス方法

- ルート：`/reflection/archive`

## 行動計画機能

### 🎯 WCMシート

WCMシートでは、Will・Can・Mustを一枚にまとめ、今後の指針や行動計画を言語化します。

#### Will（こう在りたい）

- **理想のあり方を言語化**：自分がどう在りたいかを明確にする
- WCMシートのWillテーマと連携
- 日記から価値観を抽出して自動補完

#### Can（できること）

- **現在の能力・強みを整理**：自分ができることを明確にする
- 自己診断結果と連携
- ストレングスファインダーの結果を活用

#### Must（やるべきこと）

- **現実的に取り組むべきことを明確化**：やるべきことを明確にする

## まとめ

キャリフレは、**内省伴走**をコア価値とし、AI支援により内省を習慣化し、日々の内省が「道になる」体験を提供します。

### キャリフレで実現できること

1. **自分で気づく**：診断・記録・可視化のツールで、自分の中の答えを見えるようにする
2. **「今のあなた」を否定しない**：今の環境で耐えてきたことも、守ってきたものも、ちゃんと意味がある
3. **データとして蓄積され、可視化される**：ふわっとした自己理解では終わらせない
4. **内省が「道になる」体験**：日々の内省が積み重なって、長期的な目標に接続される

### 次のステップ

1. **診断を始める**：現職満足度診断と自己診断結果を登録
2. **日記を書く**：毎日のモチベーションと出来事を記録
3. **内省チャットを始める**：AIとの対話型内省で深い内省を促す
4. **WCMシートを作成**：Will・Can・Mustを整理
5. **マイルストーンを設定**：目標とアクションを管理

**悩み続ける時間を「進んでいる実感がある時間」に置き換える**——それがキャリフレの目的です。
