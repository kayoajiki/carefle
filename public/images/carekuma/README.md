# キャリくま画像ファイル配置ガイド

このディレクトリに以下の画像ファイルを配置してください。

## 必要な画像ファイル

### 1. carekuma-full.png または carekuma-full.jpg
- **用途**: ロゴアイコン（サイドバー・ヘッダー）
- **仕様**: オレンジ背景ありの完全版
- **推奨サイズ**: 512x512px 以上
- **形式**: PNG または JPG（背景あり）
- **注意**: JPGの場合は透過背景は不要です

### 2. carekuma-icon.png
- **用途**: ヒーローセクション（トップページ）
- **仕様**: 白い円とクマのみ、背景透明
- **推奨サイズ**: 512x512px 以上
- **形式**: PNG（透過背景必須）
- **注意**: JPGは透過背景をサポートしないため、このファイルはPNG形式が必要です。JPGを使用する場合は、白い円の背景色をアプリの背景色（#F0F7FF）に合わせてください。

### 3. carekuma-favicon.ico
- **用途**: ブラウザのファビコン
- **仕様**: オレンジ背景ありの完全版
- **サイズ**: 32x32px, 16x16px（マルチサイズICO）
- **形式**: ICO

### 4. carekuma-favicon.svg
- **用途**: モダンブラウザのファビコン
- **仕様**: オレンジ背景ありの完全版
- **形式**: SVG

### 5. carekuma-apple-touch-icon.png
- **用途**: Apple Touch Icon（iOS/Android）
- **仕様**: オレンジ背景ありの完全版
- **サイズ**: 180x180px
- **形式**: PNG

## 画像の準備方法

1. 元のキャリくま画像を用意
2. 画像編集ソフトで以下を作成：
   - **完全版**: オレンジ背景（#FF6B35相当）を保持
   - **アイコン版**: 白い円とクマのみを抽出し、背景を透明化
3. 各サイズにリサイズ・最適化
4. このディレクトリに配置

## 注意事項

- ファイル名は上記の通りにしてください
- 画像が配置されていない場合、フォールバックとして元のSVGアイコンが表示されます
- ファイルサイズを最適化して、読み込み速度を考慮してください
